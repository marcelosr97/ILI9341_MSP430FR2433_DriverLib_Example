# ILI9341_MSP430FR2433_DriverLib_Example
 Example of example of ILI9341 TFT-Display (240x320) for MSP-EXP430FR2433 Launchpad Development Kit using MSP Driver Library. 

Pinout configuration:

| MSP430FR2433  | ILI9341       |
| ------------- | ------------- |
| P2.5          | SDO(MISO)     |
| 3.3V          | LED           |
| P2.4          | SCK           |
| P2.6          | SDI(MOSI)     |
| P3.2          | DC            |
| P2.2          | RESET         |
| P2.1          | CS            |
| GND           | GND           |
| 5V            | VCC           |
